# CBridge Modules

This directory contains documentation for pure C modules.

## Modules

- [file](file.md) - Filesystem utilities.
- [math](math.md) - Linear algebra and statistical functions.
- [net](net.md) - High-level network communications (UDP/TCP).
- [string](string.md) - String handling utilities.
- [vector](vector.md) - Dynamic arrays for C.
