# 📂 Source Files Hub

This directory contains the core source code for the CTool and CBridge projects.

## 📁 Directory Structure
*   `cbridge/`: Pure C module implementations and headers.
*   `ctool/`: C++11 utility classes and templates.
*   `CBridge.h / CBridge.c`: Unified entry point for the C library.
*   `CTool.hpp / CTool.cpp`: Unified entry point for the C++ library.
